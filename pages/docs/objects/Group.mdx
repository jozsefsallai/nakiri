# Group

This model defines a Nakiri group.

## Fields

| Name            | Type                                                                  | Description                                                                                                 |
| --------------- | --------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------- |
| `id`            | string                                                                | The [Snowflake ID](/docs/objects/Snowflake) of the entry.                                                   |
| `createdAt`     | date                                                                  | The date the entry was created.                                                                             |
| `updatedAt`     | date                                                                  | The date the entry was last updated.                                                                        |
| `name`          | string                                                                | The name of the group.                                                                                      |
| `description`   | string?                                                               | The description of the group.                                                                               |
| `apiKey`        | string?                                                               | The API key of the group. Not visible if the current user doesn't have the `SEE_API_KEY` member permission. |
| `creator`       | [AuthorizedUser](/docs/objects/AuthorizedUser)                        | The owner of the group.                                                                                     |
| `members`       | [GroupMember](/docs/objects/GroupMember)[]?                           | The members of the group.                                                                                   |
| `guilds`        | [AuthorizedGuild](/docs/objects/AuthorizedGuild)[]?                   | The guilds of the group.                                                                                    |
| `myPermissions` | [GroupMemberPermission](/docs/objects/GroupMember#member-permissions) | The permissions of the current user.                                                                        |
| `isCreator`     | boolean                                                               | Whether the current user is the creator of the group.                                                       |
