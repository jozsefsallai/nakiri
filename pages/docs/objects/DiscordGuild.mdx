# DiscordGuild

`DiscordGuild` is a blacklist model that stores blacklisted Discord guild IDs.

## Fields

| Name            | Type                               | Description                                                                                 |
| --------------- | ---------------------------------- | ------------------------------------------------------------------------------------------- |
| `id`            | string                             | The [Snowflake ID](/docs/objects/Snowflake) of the entry.                                   |
| `createdAt`     | date                               | The date the entry was created.                                                             |
| `updatedAt`     | date                               | The date the entry was last updated.                                                        |
| `deletedAt`     | date                               | The date the entry was deleted.                                                             |
| `guildId`       | string, null                       | The Discord guild ID the entry belongs to. If null, the entry is not restricted to a guild. |
| `group`         | [Group](/docs/objects/Group), null | The group the entry belongs to. If null, the entry belongs to the global blacklist.         |
| `severity`      | [Severity](/docs/objects/Severity) | The severity of the entry.                                                                  |
| `name`          | string                             | A friendly name for the entry to make it easier to identify.                                |
| `blacklistedId` | string                             | The blacklisted Discord guild ID.                                                           |
